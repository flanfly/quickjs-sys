# quickjs-sys

Low-level Rust bindings for the [QuickJS Javascript Engine](https://bellard.org/quickjs/).

```toml
# Cargo.toml
[dependencies]
quickjs-sys = "0.1"
```

The documentation of the C API can be found
[here](https://bellard.org/quickjs/quickjs.html).
